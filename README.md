## 基于Java实现的玩具RPC
## 简介
项目基于[@throwable](https://github.com/zjcscut)的[Netty和SpringBoot实现一个轻量级RPC框架](https://github.com/zjcscut/netty-tutorials)，并添加了zooKeeper
## 技术栈
springBoot + zooKeeper + netty

- [x] 序列化、反序列化
- [x] 创建调用对象、调用rpc方法
- [x] 服务注册与发现
